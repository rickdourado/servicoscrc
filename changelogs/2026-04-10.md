# Changelog 2026-04-10

## Backend
- **Contract Analysis Workflow**: `app.py` update. Added endpoints `/api/anonymize` (extraction/masking) and `/api/analyze-text` (AI analysis) for the new Contract Analysis workflow.

## Frontend
- **Contract Analysis**: implemented a two-step analysis flow in `analisecontrato.html` containing a mandatory preview/edit area for anonymized text before final AI submission.
- **Wireframes Restoration (AS-IS)**: Corrected missing dropdown options (`Categoria`, `Tipo de Solicitante`, `Origem da Ocorrência`, `Prioridade`) across four local wireframe files (`irregular_veiculo`, `wireframe_meio_ambiente`, `wireframe_sossego`, `wireframe_veiculo_abandonado`), adhering strictly to the properties mapped in reference files.
- **Form Viewer Upgrade**: Inserted a cache-busting timestamp parameter (`?t=timestamp`) natively into the `iframe` injections inside `analiseformulario.html` to prevent aggressive browser caching issues that blocked UI visual updates.
- **Fiscalização de Obras**: Moved `wireframe_fiscalizacao_obras.html` from `refs/wireframes` to `frontend/wireframes` and added it natively to the analysis panel's select dropdown.

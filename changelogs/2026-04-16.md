# Changelog - 2026-04-16

## 🚀 Novas Funcionalidades e Melhorias

### 🏛️ Identidade Visual e Branding
- **Novo Header e Footer**: Implementação de moldura institucional baseada no portal 1746.rio em todas as páginas.
- **Identificação Oficial**: Atualização do nome para **Coordenadoria de Relacionamento com o Cidadão (CRC)**.
- **Ativos Oficiais**: Integração do logo oficial do 1746 e link direto para o portal de ajuda ("Visitar Site Oficial").
- **Refinamentos de UI**: Simplificação do título do dashboard para "Automatização de Tarefas" e melhoria em botões de ação.

### 🛡️ Segurança e Anonimização
- **Módulo de Anonimização**: Criação do script `backend/scripts/anonymize_pdf.py` para remoção automática de dados sensíveis (CPF, CNPJ, nomes próprios e institucionais) em contratos PDF.
- **Remoção de Imagens**: Processamento visual para cobrir assinaturas e imagens embutidas em documentos.
- **Privacidade**: Limpeza de termos de privacidade temporários para focar na descrição técnica do serviço.

### 🧠 Inteligência Artificial e Contexto
- **Análise de Relatórios**: Evolução da ferramenta de análise de contratos para suportar a análise de **Relatórios Mensais de Execução**.
- **Contexto de Contrato Base**: Implementação de injeção automática do contrato anonimizado como base de conhecimento para a IA.
- **Extração Markdown**: Criação do script `backend/scripts/extract_contract_md.py` para converter PDFs em Markdown estruturado, otimizando o processamento pelo Gemini.
- **Gestão de Prompts**: Reorganização dos prompts em pastas específicas por contrato (`backend/prompts/czrm/`).

### 🛠️ Backend e API
- **Endpoint Dinâmico**: Nova rota `/api/contracts` para listar contratos disponíveis para análise.
- **Refatoração do Carregamento**: Suporte a subdiretórios de prompts conforme o contrato selecionado.

# Changelog - 2026-04-12

## 🚀 Novas Funcionalidades / Scripts
- **Extração de Planilhas AS-IS**: Adicionado o script `extract_fields_to_excel.py` em `backend/scripts/` para automação de extração de campos em wireframes HTML legados (AS-IS).
  - Utiliza `beautifulsoup4` para leitura das tags e identificação de campos (título, descrição, tipo, obrigatoriedade, seções).
  - Utiliza `openpyxl` para estruturar e salvar os dados no formato `.xlsx`.
  - Diferencia cada HTML processado usando abas (sheets) dedicadas na planilha final gerada.

## 🛠️ Resumo Técnico (Backend)
O processamento varre a pasta `refs/wireframes` verificando pela classe `row` ou `two-col` e mapeando rótulos (`<label>`). O status "Obrigatório" é validado capturando classes do escopo (`req`, `req-blue`). Todo o processo é executado através de virtual env pelo `uv run`.

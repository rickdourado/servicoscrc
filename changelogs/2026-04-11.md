# Serviços CRC - Resumo de Atualizações

## Data: 10 de Abril de 2026 (Ref. Madrugada de 11 de Abril de 2026)

### ✨ Adições:
- Adicionado script Python `run.py` na raiz do projeto para servir como alternativa multiplataforma (Windows/Linux/Mac) ao `start.sh`. O script roda processos paralelos para servir a API de backend (FastAPI/uvicorn na porta 8000) e o servidor frontend HTML (na porta 3000), gerenciando também um desligamento limpo de ambos os processos via SIGINT (Ctrl+C).

### 🛠 Alterações:
- O uso de ambiente continua priorizando `uv`, conforme as regras locais de arquitetura.
